How it works

Hebrew characters → converts to Latin. Latin characters → converts to Hebrew. Same shortcut, always works.

⚡

Frequently asked questions

Is it safe? Can it read everything I type? ▾
No. The tool only activates when you press Ctrl+Shift+H. It does not monitor your keyboard at all.
Does it send my text to the internet? ▾
Never. The conversion is a simple character lookup table. It runs entirely on your Mac. Zero network calls.
Why does it need Accessibility permission? ▾
macOS requires this for any tool that simulates keystrokes (Cmd+C, Cmd+V). It's a Mac security feature.
What is Hammerspoon? ▾
Hammerspoon is a free open-source Mac automation tool. Our script is its config file - readable on GitHub.
Does it work with German and English? ▾
Yes. The mapping is based on physical key positions, not language. Works for both German and English.
How do I uninstall it? ▾
Delete ~/.hammerspoon/init.lua and quit Hammerspoon. Or: brew uninstall --cask hammerspoon
